# AutoChord

AutoChord是一个为给定旋律配和弦的项目。

### 计划表

初步计划为三部分：

- 基本复现论文中用BiLSTM学习的模型
- 转为类SAT问题，或者序贯决策模型
- 用BiLSTM学出的模型评价每一步可选和弦分数，用贪心法序列生成和弦（主要解决深度模型难以考虑非法进行）

#### 进程表

具体需要完成的任务：

- 数据预处理，所有的数据处理成标准numpy格式并存储，其中需要对和弦进行识别，并统一拉到C大小调上，写一个规则判断函数以判定是否违规（规则集应当可扩充）；另外定义一个序列罚函数，用于神经网络的训练，以及最后的贪心法
- 深度模型搭、搭建、模型保存标准化流程
- 待续
